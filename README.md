# utube-AI-tools
Tools to help with you tube videos 

All you need is an open ai api key to get started.....
Paste key in to field and off you go

img.py - creates an image for a guven prompt

mp3.py creates an mp3 using an mp4 as input

multi.py enables multi thread mp3 s for stuff like radio station or interview , discussion etc

rewrite.py takes a transcript and re writes it

script.py writes a u tube script on any subject

search.py  searches the web and downloads images for a given keyword or phrase

t2s3.py  text to speech
